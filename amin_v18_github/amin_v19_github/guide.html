新手教學

快速入門阿民天堂，從下載、創角到加入血盟。

第一步：下載登入器

前往下載專區取得登入器，解壓縮後執行啟動程式。

第二步：加入社群

加入 LINE 或 Discord，領取新手禮包與即時公告。

第三步：完成新手任務

建議先完成新手地圖任務，取得基礎裝備後再挑戰 BOSS。

新手推薦路線

  1. 創角後先領取新手禮包。
  2. 練功到指定等級後前往主城補給。
  3. 選擇合適魔法娃娃：近戰紅火、防禦黃燄、遠攻綠光、法師藍炙。
  4. 加入血盟參與攻城與世界 BOSS。
