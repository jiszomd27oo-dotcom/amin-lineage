每日簽到 / 投票獎勵

每天上線領取福利，累積簽到再拿限定道具。

每日簽到

每日登入遊戲後，可於活動 NPC 領取補給禮盒。

獎勵：金幣、藥水、強化卷軸
連續簽到

連續 7 天、14 天、30 天可領取階段性獎勵。

獎勵：娃娃材料、耳環材料、活動幣
社群分享

加入 LINE / Discord 並分享活動貼文，可領社群禮包。

獎勵：新手補給箱、祝福卷軸

領取流程

  1. 登入遊戲並確認角色在線。
  2. 前往主城尋找「活動管理員」。
  3. 選擇每日簽到或連續簽到獎勵。
  4. 獎勵會直接發送至角色背包。
