站長更新中心

不用改複雜程式，只要修改 data 資料檔，就能更新首頁公告、排行榜與伺服器資訊。

更新流程

  1. 進入 GitHub Repository：amin-lineage
  2. 點開 data 資料夾
  3. 選擇要修改的 JSON 檔案
  4. 點右上角鉛筆 Edit
  5. 修改內容後按 Commit changes
  6. 等待 Cloudflare 約 30 秒自動部署

公告資料：data/news.json

用來控制首頁與公告區內容。

[
  {
    "date": "07/07",
    "type": "公告",
    "tag": "red",
    "title": "阿民天堂正式開服"
  }
]

伺服器資訊：data/server.json

用來控制首頁左側伺服器狀態。

{
  "name": "阿民天堂 2.7",
  "status": "運行中",
  "exp": "20 倍",
  "drop": "2 倍",
  "adena": "2 倍",
  "weapon": "2 倍",
  "online": "386 人"
}

排行榜資料：data/ranking.json

可修改等級、PK、財富、血盟、BOSS 排行。

{
  "level": [
    ["宋家大佬", "Lv89"],
    ["基努李維", "Lv88"]
  ],
  "pk": [
    ["辣手", "128勝"],
    ["暴力美學", "96勝"]
  ]
}

注意事項

  • JSON 的逗號、雙引號不能刪錯。
  • 每一筆資料最後一行不要多加逗號。
  • 修改前可以先複製備份原本內容。
  • 如果網站沒有更新，請等待 Cloudflare 部署完成後重新整理。
