帳號安全中心

官方不會索取密碼，不會要求玩家提供遠端控制或私下交易帳號。

防詐騙提醒

  • 請勿將帳號、密碼交給任何人。
  • 客服只會透過官方 LINE / Discord 聯繫。
  • 不購買來源不明的遊戲幣與道具。
  • 遇到冒充 GM 請立即截圖回報。

帳號保護建議

  • 密碼請勿與其他遊戲或論壇重複。
  • 定期更換密碼。
  • 網咖遊玩後務必完整登出。
  • 下載檔案請只使用官方網站入口。

官方聯絡

發現異常登入、裝備遺失、假客服或外掛疑慮，請到 GM 回報中心提交資料。
